Plug-in Info & Activation

The bottom half of the window displays one of two tabs. The Info tab contains a description of what the plug-in does and the commands you can use to activate it. Most plug-ins start out using the Control key, combined with the backtick (`) or a number, but this may differ based on your keyboard layout.

While OmniDazzle is open, it watches for these commands, no matter which application you're using. Just work on your Mac normally, and when you want an effect, use one of the commands.

If you want to change a command to use certain keys, certain mouse buttons, or a shake of the mouse, click the command in the Info tab to be taken to the Configuration tab where you can customize the commands.

OmniDazzle Release Notes

Here you'll find important information about new builds.

1.1.1 - Aug 26, 2009
  • Snow Leopard Compatibility
    • Fixed compatibility issues with the built-in crash catcher.
  • Features
    • The built-in software update now offers you the option to ignore a specific update.
 1.1 - Feb 24, 2009
  • OmniDazzle is now free!
 1.0.2 - Nov 14, 2007
  • Leopard Compatibility
    • Fixed all known compatibility issues.
  • Interface
    • Hot keys that conflict a system shortcut in the Keyboard & Mouse preferences show up in red in OmniDazzle's interface. Sadly, the Spaces hotkeys aren't reported by the system API. We've logged a bug with Apple to fix this.
